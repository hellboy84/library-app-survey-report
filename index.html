6. 考察

主要な発見

  1. 図書館利用頻度と年代の関係：
    • 10代は高頻度利用者が多く（週に数回以上：34.7%）、図書館を積極的に活用
    • 20代・40代は低頻度利用者が多い（それ以下が約40%）
    • 60代以上は利用頻度が幅広く分散し、高頻度利用者も一定数存在
  2. デジタルネイティブ世代の積極的利用： 10～30代は高い利用意向（90％以上）と多機能利用（平均4機能以上）を示している
  3. 50代以上のデジタルデバイド： 50代以上では利用機能数が大幅に減少（1～1.3機能）し、利用意向も50％前後に留まる
  4. 満足度の年代格差： 20代が最高満足度（4.0）を示す一方、50代は最低（2.17）で、年代による体験格差が存在

図書館利用頻度と アプリ利用意向の関係

  • 10代：図書館を頻繁に利用し、アプリへの関心も高い（理想的なユーザー層）
  • 20代：図書館利用は少ないが、アプリへの満足度・利用意向は高い（潜在的ユーザー層）
  • 60代以上：図書館は積極的に利用するが、アプリ利用への障壁が存在

示唆と提言

  1. 高齢者向けUI/UX改善： 図書館を頻繁に利用する60代以上のユーザーにもアプリを利用してもらうため、シンプルで直感的なインターフェースの導入が必要
  2. 働き盛り世代への訴求： 図書館利用頻度は低いが、アプリへの関心は高い20代～40代に向けて、図書館利用を促進する機能（オンライン予約、電子書籍など）の充実が有効
  3. 年代別アプローチ：
    • 10代：現在の高い利用頻度・満足度を維持する機能強化
    • 20～40代：図書館利用促進とアプリ活用の両立
    • 50代以上：段階的なデジタル化支援とサポート体制
  4. 利用パターンに応じた機能設計： 低頻度利用者向けには利便性重視の機能、高頻度利用者向けには高度な機能を提供する差別化戦略

今後の課題

  • 図書館利用頻度の低い世代への利用促進策
  • 50代以上のデジタルリテラシー向上支援
  • 年代間の満足度格差解消
  • 図書館利用とアプリ利用の相乗効果創出

まとめ

本調査により、図書館利用頻度とアプリ利用意向に明確な年代差が存在することが判明した。 特に、図書館を頻繁に利用する層（10代・60代以上）とアプリに関心の高い層（10～40代）の特性を理解し、 それぞれに適したサービス設計が重要である。図書館利用とアプリ利用の相乗効果を生み出すことで、 全年代にとって価値のあるサービスを提供できると考えられる。
